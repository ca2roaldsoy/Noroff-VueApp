# puppyrecipe

## Run Project
Clone the Repo:
```
git clone https://github.com/ca2roaldsoy/Noroff-VueApp.git
```
Run in terminal:
```
npm install
```

### Compiles and hot-reloads for development
Run in terminal:
```
npm run serve
```
Run in browser:
```
localhost:8080
```

### Compiles and minifies for production
Run in terminal:
```
npm run build
```

### Lints and fixes files
Run in terminal:
```
npm run lint
```
